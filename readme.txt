== Description ==

GMO Share Connection plugin is designed for easy social sharing by letting user choose place/pages to use icons.
9 social network services are supported in this plugin including Facebook and Twitter.

== Installation ==

= Manual Installation =

1. Upload the entire `/gmo-share-connection` directory to the `/wp-content/plugins/` directory.
2. Activate GMO Share Connection through the 'Plugins' menu in WordPress.
